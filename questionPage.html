    • Why my main query is running successfully at the same time as subquery returns an error?

    select 1/0 as test query returns an error "Divide by zero error encountered" But this one works fine: select count(*) from (select 1/0 as test) subquery Could anyone explain why it happens?

    Asked by: Herman Meville

    Viewed: 1819 times

    Voted: 1891 times

    1 Answers

    It works, because sql server doesn't execute 1/0 from the subquery. You only want to count rows, sql server is smart enough to know that it is an additional action that is excess. For example these two statements don't work, because in these cases 1/0 must be executed.
    The main query interacts with the subquery as normal SQL statement while the subquery is a purely mathematical problem which is you can not divide by zero.
